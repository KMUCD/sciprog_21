magicsquare.c
In this c script we check if a 3 by 3 square is magic. It is magic if the rows columns and diagonals sum to some constant c for all rows, columns and diagonals. We also call a header file magicsquare.h which checks if our square is magic.

To compile write "gcc -o magicsquare magic_square.c"



magic_square_fileread.c
In this function, the above script is repeated except we read in txt files using fscanf instead of using scanf to input our magic sqaure files

To compile write "gcc -o magicsquare_fileread magic_square_fileread.c"



hanoi.c
In this c script we solve the tower of hanoi problem recursively.

To compile write "gcc -o hanoi hanoi.c"
