use_printf.c
In this function we demonstrate printing using printf. We print two integers using %d and two floats using %.xf

To compile write "gcc -o use_printf use_printf.c". No input required


trapezoidal.c
Compute the integral via the Trapezoidal Rule approximation.
We compare the values of the trap rule integral of tan(x) from 0 to pi/3
and the actual value of this integral, log(2).

To compile write "gcc -o trap trapezoidal.c" No input required.
